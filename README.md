# Autojs_Sign-in

#### 介绍
基于Auto.js开发的Android自动签到APP

已支持：

1.掌上道聚城签到、每日任务（部分基于坐标，适配3200*1440）

2.小米社区签到

3.掌上穿越火线签到

#### 使用说明

下载安装app，启用无障碍服务。

配合李跳跳跳过广告使用更佳，附自用规则、李跳跳apk备份。
